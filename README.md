# Class Registration System
## Instructions
This is a class registration system. First of all you need to input an existing student information for example Alice, 12345 to add your self as a new student into the system. 


<img width="357" alt="截屏2024-04-15 22 06 58" src="https://github.com/Jingpeng0032/Semester-project-2024/assets/160645754/78844c02-11ff-4df5-a4d5-e89aad00fa1f">

After you have input student information, the system will display greetings and available actions

<img width="677" alt="截屏2024-04-15 22 07 12" src="https://github.com/Jingpeng0032/Semester-project-2024/assets/160645754/90b3ae9a-d14c-43cd-a6c8-43285e95d937">

If you want to enroll in a course, enter 1, drop a course, enter 2, add a student, enter 3. If you want to quit the program, enter 4. This is where you should add yourself as a new student to the system. The system will ask you to create your name ID, and your major. After the system display 'New student added', enter 'q' to quit the program.

<img width="592" alt="截屏2024-04-15 23 18 49" src="https://github.com/Jingpeng0032/Semester-project-2024/assets/160645754/e0382916-2230-46a3-86a8-881895a51c1f">

## Rules of registration
a) Each student need to have at least 16 credits, at most 18 credits every semester.  

b) Students can only choose Major Courses that belongs to your major. There is no limitation for Elective Courses.  

c) Students need to complete all the prerequisites of the chosen course before enroll in this course  

d) Each course section has a maximum number of students of 50
